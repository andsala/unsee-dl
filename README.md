# Usage

Requires [Mozilla Firefox](https://www.mozilla.org/it/firefox/) and [geckodriver](https://github.com/mozilla/geckodriver).

```
pip install -r requirements.txt

python unsee-dl.py [-o OUT_DIR] <id...>
```
